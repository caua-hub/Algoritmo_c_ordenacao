                                                      PROJETO:
----------------------------------------------------------------------------------------------------------------------
O projeto consiste em um sistema de busca de valores em que eh
possivel a insercao de um arquivo .txt pelo usuario contendo
numeros inteiros. O sistema permite que o usuario faca uma busca
linear e binaria, e tambem permite a ordenacao do vetor se necessario.
----------------------------------------------------------------------------------------------------------------------
                        BUSCAS:
LINEAR: Eh possivel fazer a busca linear sem a necessidade de ordenacao.
BINARIA: Para realizar a busca binaria, eh necessario que o vetor esteja ordenado.
----------------------------------------------------------------------------------------------------------------------
                        ORDENACAO:
Se necessario ordenacao, o usuario devera escolher entre as opcoes de vetores do programa.
BUBBLE SORT; INSERTION SORT; SELECTION SORT; MERGE SORT; QUICK SORT; RADIX SORT.
----------------------------------------------------------------------------------------------------------------------
                        RELATORIO:
Para criar um relatorio, o vetor precisa estar carregado no programa e ter realizado um tipo de ordenacao.
Depois de gerado, o arquivo ficara disponivel na pasta 'output' da pasta corrente com o nome: relatorio_ti_ti.txt.
